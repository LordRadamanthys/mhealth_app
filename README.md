# MHealth

<a href="https://www.linkedin.com/in/mateuslm/" target="blank"><img src="https://user-images.githubusercontent.com/49004830/105938894-e156e180-6036-11eb-993d-33da625f10dd.png" width="150" height="100"/></a>
> In 2020 with the coronavirus pandemic my concern for health increased and I noticed that I have no organization about my health history, medications, tests, vaccines etc. So I decided to create an app where I could put all this information.

***MHealth*** serves as a system to manage your medical records, vaccines, exams and even gym exercises. You can upload documents to keep the information already registered more complete.

# Technologies

  - React Native (EXPO)
  - Axios
  - API build with python using Flask
  - DB in MySQL
  
# Features
  - Login
  - Create user
  - forgot password
  - add exams, vaccines, gym, training and medicines
  - add files to exam and vaccines

# Future features
  - Login with fingerprint
  - Push notification to remember when you have to return to exam or vaccine
  - Integration with smart watch and Amazon Alexa
